#### 数据结构与算法

##### 必记知识点
对于树的计算，记住以下三个基本公式即可，其余性质都可以推出来：  
1. 任何非空树中：分支数 = 所有结点度之和 = n-1  
2. 度为m的树中：n = n0 + n1 + … + nm  
3. 度为m的树中：所有结点度之和 = n1 + 2n2 + … + mnm
