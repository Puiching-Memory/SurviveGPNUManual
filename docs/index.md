# 广师大生存手册 survive GPNU manual

## 序章

本项目灵感来源于[上海交通大学生存手册](https://survivesjtu.gitbook.io/survivesjtumanual)，针对广师大学子进行了本地化，同时也推荐未入学和高考报考志愿的朋友阅读。  
该项目维护者仅由本校学生组成,**无任何利益关系**,经供参考。

## 目标
**本文档目标不是替代任何官方博客,而是由校内校外曾经生活过的同学所记录的生活经验**

对于正文页面，要求：  
1. 不要重复搬运已有的文章，直接提供源链接即可  
2. 对于不了解的事情，不应该草率编写  
3. 内容应避免过多主观观点  
4. 经久不衰的文章价值在于提供方法论，做信息整合是不可持续的，只能寄托于社区贡献

对于博客页面，要求：  
1. 搬运前必须与原作者联系  
2. 媒体文件必须控制大小
