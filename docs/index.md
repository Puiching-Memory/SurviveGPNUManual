# 欢迎使用 FastAPI React Starter 文档

本文档提供了基于 FastAPI React Starter 模板理解、设置、开发和部署项目的综合指南。

此启动模板旨在为使用 Python (FastAPI) 后端和 React 前端构建 Web 应用程序提供现代化、功能齐全的基础，利用最佳实践和流行工具。

## 主要功能

- **后端 (FastAPI)：** 强大且高效，具有异步 SQLAlchemy、JWT 身份验证、基于角色的访问控制、结构化日志记录等功能。
- **前端 (React 19)：** 使用 TypeScript、Vite、Tailwind CSS、shadcn/ui 组件、客户端路由和自定义钩子的现代 React。
- **容器化环境：** 使用 Docker Compose 轻松设置和一致的开发/生产环境。
- **数据库管理：** 集成的 Alembic 迁移用于模式管理。
- **自动化设置：** 用于快速环境准备的便捷脚本。
