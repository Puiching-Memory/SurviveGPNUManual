# 社区

## 想要分享你的故事？马上开始！

第一步：注册一个Github账号

第二步：访问源代码仓库

第三步：创建一个分支

第四步：写上你自己的故事

第五步：发起合并请求

!!! 注意
    不要直接下载zip存档，因为其中不包含.git文件，将无法使用git命令发起合并请求

## 发现文章错误？提出问题！

第一步：注册一个Github账号

第二步：前往issue板块

第三步：查找是否有人已经提交相同的问题

第四步：创建一个新问题

## 可视化编辑

```

conda activate gpnu
pip install -r requirements.txt
```

